Общая информация:
В директории содержатся скрипты и DAG для ETL, а также файлы с моделями для SQLAlchemy.

В DAGs хранятся сами графы, в models модели, а в scripts код тасков. Один DAG может не соответсвовать одному файлу скриптов.

2024-11-07
Ограничения по git clone.
Директория не подразумевает цельную рабочую систему и предназначена для хранения скриптов. При загрузке директории в текущем виде, DAG и таски работать не будут, так как будут возникать ошибки импорта.
В дальнейшем импорты тасков в DAG будут переписаны.

Так как директория является публичной, пакет access_, используемый в скриптах, не публикуется.

Документация к скриптам будет добавлена позже.
